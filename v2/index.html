About Jeremy Huang

關於我

黃俊豪

Jeremy

1991 生於台北，台灣

一個曾經以為會成為平面設計師卻誤入歧途的工程師。大學就讀文化大學資訊傳播系時，在學群分組時不經思考就選擇設計組的我，卻因指導教授的邀請，一頭栽進了程式的世界。從 Java 開始寫起，寫進了 Android ，觸碰了基本的網頁，到後來的後端程式，一步步的越寫越有興趣，甚至於學生時期就開始慢慢地展開接案生涯。

進入職場後，各式各樣的專案讓人頭暈目眩，卻也接觸了更多的領域，視訊串流、金流、物流、物聯網等等的。活到老學到老這句話套用在工程師身上更是符合，每天都在學習新的事物、新的架構、新的語言、新的概念，至今仍認為自己是個小菜鳥的工程師。

目前專注於以 Kotlin / Java 開發 Android App ，網頁前端則採用基本的 Bootstrap 框架，後端程式則使用 C# .net 居多，但因開發 WordPress 樣式，簡單的 php 也還是可以的，資料庫的部分則較常使用 MSSQL, MySql, MongoDB 等。
